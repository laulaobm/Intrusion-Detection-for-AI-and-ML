The internet has been around for a long time now, and there is still an ongoing war between hackers and security experts. 
Current network security measures are not sufficient to protect against all types of cyber attacks, and there is a need for newer solutions.
\nextparagraph
This is why there is a growing interest in the field of cybersecurity in using Machine Learning (ML) techniques to automatically detect and prevent certain types of cyber attacks. Since we study in the field of Computer Science and are generally interested in data security, we naturally hear a lot about different cyber attacks happening worldwide, like the recent Deezer attack \cite{purplesec_deezer_data_leak} that saw the data of 228 million users being leaked. 
\nextparagraph
However, detecting cyber attacks is not a simple task, and there are a lot of different types of attacks, each with their own characteristics and patterns. We already learned about some of these attacks in some of the courses we all took, like ``System Security'', but we wanted to learn more about how to automatically detect these attacks using ML approaches. We are all interested in the inner functioning of an Intrusion Detection System (IDS), and we want to learn more about how to build one using ML techniques.

Unfortunately, there is no single solution to detect all kinds of attacks, and more sophisticated attack patterns are emerging every day. But there are still general patterns found among similar types of attacks, and those can be detected using ML techniques. There is already a lot of research being done in this field and many promising results are being shown, but there is still a lot of work to be done and new attacks are emerging every day. 

Therefore, we propose the following hypothesis: 
``ML techniques can be used to automatically detect certain types of cyber attacks, and can be an effective tool in the fight against cyber crime.''

Taking this hypothesis into account, the goal/outcome of our project consists of two parts: 
\begin{itemize}
    \item To build a model that gives us the ability to automatically detect certain types of cyber attacks, and to evaluate its performance using already established approaches as comparisons. 
    \item To analyze the results and draw conclusions about the effectiveness of ML techniques in detecting cyber attacks, and to identify potential areas for improvement.
\end{itemize}
